# good_kensaku

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### chrome拡張機能を実際に動かす
1. `chrome://extensions`にアクセス
2. デベロッパーモードをON
3. パッケージ化されていない拡張機能を読み込むをClick
4. `dist/`以下を選択
5. `good kensaku`をクリックして有効化
